```

в”Ӯ   в”ңв”Җв”Җ rustledger-loader/       # File loading, includes, processing pipeline
в”Ӯ   в”ңв”Җв”Җ rustledger-booking/      # Interpolation and booking engine (7 methods)
в”Ӯ   в”ңв”Җв”Җ rustledger-validate/     # Validation rules (28 error codes)
в”Ӯ   в”ңв”Җв”Җ rustledger-query/        # BQL query engine
в”Ӯ   в”ңв”Җв”Җ rustledger-plugin/       # Native + WASM + Python plugin system
в”Ӯ   в”ңв”Җв”Җ rustledger-plugin-types/ # WASM plugin interface types
в”Ӯ   в”ңв”Җв”Җ rustledger-importer/     # CSV/OFX bank statement import
в”Ӯ   в”ңв”Җв”Җ rustledger-lsp/          # Language Server Protocol
в”Ӯ   в”ңв”Җв”Җ rustledger-wasm/         # WebAssembly bindings for JS/TS
в”Ӯ   в”ңв”Җв”Җ rustledger-ffi-wasi/     # FFI via WASI JSON-RPC
в”Ӯ   в””в”Җв”Җ rustledger/              # CLI tools (rledger, bean-* commands)
в”ңв”Җв”Җ packages/
в”Ӯ   в”ңв”Җв”Җ vscode/                  # VS Code extension (LSP client)

в””в”Җв”Җ spec/                        # Specifications (this folder)
```
